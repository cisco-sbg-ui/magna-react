## Usage

### Import

<ComponentImport components="useABreakpoint" />

### Thresholds

Media queries are set for the following breakpoints:

- xs: 479
- sm: 767
- md: 960
- lg: 1280
- xl: 1680
- xxl: 2080

These are exported as an object named `breakpointThresholds`

### Usage
