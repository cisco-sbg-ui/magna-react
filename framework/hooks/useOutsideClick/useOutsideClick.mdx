## Usage

### Import

<ComponentImport components="useOutsideClick" />

### Description

`useOutsideClick` detects when the user is clicking outside of a specified element in the DOM.

### Parameters

A configuration object with the following keys:

- `isEnabled`: If the hook should listen for outside clicks of your element
- `rootRef`: The React Ref in which clicks outside of this element will register as an outside click
- `containerRef`: An optional React Ref to serve as an upper boundary to limit "how far" out the outside clicks should be detecetd from you `rootRef`
- `onClick`: A function to be called when the user clicks outside of `rootRef`, and if `containerRef` is passed, within it, too
