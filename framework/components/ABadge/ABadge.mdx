  withTwoPanes

#### Badge Sizes

<Playground
  code={`<>
  {["small", "medium", "large"].map((size) => {
    const sizeProp = {
      [size]: true,
    };
    return (
      <ABadge {...sizeProp} content="3" className="mr-4">
        <AButton icon>
          <AIcon size={48}>bell</AIcon>
        </AButton>
      </ABadge>
    );
  })}
</>;
`}
/>

#### Badge Statuses

<Playground
  code={`<>
  {["error", "info", "success"].map((level) => {
    return (
      <ABadge level={level} content="99+" large className="mr-4">
        <AButton tertiaryAlt icon className="pa-1">
          <AIcon size={48}>bell</AIcon>

      </ABadge>
    );
  })}
</>
`}
/>

<Playground

## Accessibility Notes

By default, `ABadge` components are assigned the [WAI-ARIA](https://www.w3.org/WAI/standards-guidelines/aria/) role of [alert](https://www.w3.org/TR/wai-aria/#alert) which denotes that the alert "is a live region with important and usually time-sensitive, information."
