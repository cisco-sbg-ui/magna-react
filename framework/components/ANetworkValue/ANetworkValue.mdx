# Network Value

Use for MAC addresses, IP addresses, networking IDs

## Install

Integrate with your build to [auto-import](/#integrating), or add an import in your component:

```js
import {ANetworkValue} from "@cisco-sbg-ui/magna-react";
```

## Usage

<Playground
  code={`() => {
