# Panels

## Install

Integrate with your build to [auto-import](/#integrating), or add an import in your component:

```
import {
  APanel,
  APanelHeader,
  APanelFooter,
  APanelBody,
  APanelTitle
} from "@cisco-sbg-ui/magna-react";
```
