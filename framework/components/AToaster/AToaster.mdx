# Toasts

## Install

Integrate with your build to [auto-import](/#integrating), or add an import in your component:

```
import {useAToaster} from "@cisco-sbg-ui/magna-react";
```

## Accessibility Notes

The toaster applies the [aria-live](https://www.w3.org/TR/wai-aria/#aria-live) attribute with a value of `polite` to alert the user of the new content.

Also applied is the [WAI-ARIA](https://www.w3.org/WAI/standards-guidelines/aria/) role of [status](https://www.w3.org/TR/wai-aria/#status) to indice it is a `live region`.
