# Cards

## Install

Integrate with your build to [auto-import](/#integrating), or add an import in your component:

```
import {
  ACardContainer,
  ACardBasic,
  ACardItem,
  ACardHeader,
  ACardTitle,
  ACardSubTitle,
  ACardContent,
  ACardFooter
} from "@cisco-sbg-ui/magna-react";
```

## Container card

Used to group content and actions into logical sections or buckets, it can contain [Basic cards](/components/card#basic-card).

<br />

<Playground
  code={`
<ACardContainer>
  <ACardHeader>
    <ACardTitle>Hi I'am Container</ACardTitle>
  </ACardHeader>
  <ACardContent>
    <div style={{background:'rgba(0,0,0,0.05)', padding:10}}>Sample Content</div>
  </ACardContent>
</ACardContainer>
`}
/>

### Anatomy of container

- Title / Subtitle are optional
- Content area can be single or multi-column layouts
- Buttons are optional and are typically positioned to the right in the bottom or top of the container.

<br />

## Basic card

Cards group related information in a flexible-size containers.
Basic cards typically contain a few short, related pieces of information and can be
interactive or non-interactive.
For page level groupings, use [Container Cards](/components/card#container-card).

### Anatomy of card

- Title / Subtitle are optional
- Content area can be single or multi-column layouts
- Buttons are optional and are typically positioned to the right in the bottom or top of the container.
