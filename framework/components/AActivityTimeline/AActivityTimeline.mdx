#### Collapsible Usage (Uncontrolled)

To make a timeline item collapsible, pass the `defaultCollapsed` prop, which will make the content you pass to `AActivityTimelineItemBody` toggle on timeline item clicks.

If you have more sophisticated requirements for toggling the timeline items, you can control the collapsed state yourself (see documentation further down the page.)

  code={`<AActivityTimeline>

#### Collapsible Usage (Controlled)

If you have special requirements for collapsing timeline items, you can control the state yourself. For example, suppose that only one timeline item can be opened at a time. In that case, you can pass the `isCollapsed` and `onCollapse` props to the `AActivityTimelineItem` component:

<Playground
  code={`() => {
  const [openedIndex, setOpenedIndex] = useState(0);
  return (
    <AActivityTimeline>
      <AActivityTimelineItem
        isCollapsed={openedIndex !== 0}
        onCollapse={() => setOpenedIndex(0)}
      >
        <AActivityTimelineItemHeader>
          <AActivityTimelineItemTitle subtitle="2022-02-12 21:11:10 UTC">
            Account created
          </AActivityTimelineItemTitle>
        </AActivityTimelineItemHeader>
        <AActivityTimelineItemBody>
          Your account was created under the username "zezima".
        </AActivityTimelineItemBody>
      </AActivityTimelineItem>

      <AActivityTimelineItem
        isCollapsed={openedIndex !== 1}
        onCollapse={() => setOpenedIndex(1)}
      >
        <AActivityTimelineItemHeader>
          <AActivityTimelineItemTitle subtitle="2022-02-14 21:12:10 UTC">
            Tutorial island completed
          </AActivityTimelineItemTitle>
        </AActivityTimelineItemHeader>
        <AActivityTimelineItemBody>
          You finished the tutorial. Welcome to Gielinor!
        </AActivityTimelineItemBody>
      </AActivityTimelineItem>

      <AActivityTimelineItem
        isCollapsed={openedIndex !== 2}
        onCollapse={() => setOpenedIndex(2)}
      >
        <AActivityTimelineItemHeader>
          <AActivityTimelineItemTitle subtitle="2022-02-14 22:11:10 UTC">
            Cook's Assistant started
          </AActivityTimelineItemTitle>
        </AActivityTimelineItemHeader>
        <AActivityTimelineItemBody>
          You have started the first ever quest released in the game, which was
          one of the only six quests available during the game's initial launch.
          In this quest, you'll help the Cook of Lumbridge Castle make a
          birthday cake for Duke Horacio.
        </AActivityTimelineItemBody>
      </AActivityTimelineItem>
    </AActivityTimeline>

);
};`}
/>

#### Dividers

By default, dividers only render when the timeline item is collapsible. However, if you need to render non-collapsible items with dividers, then you can pass the `withDivider` prop to `AActivityTimelineItem`.
