#### Sizes

`AButton` supports the small and medium size options. The `medium` prop is encouraged in the Magnetic Design System.

<Playground
  code={`<>
  <AButton small className="mr-2">
    Button
  </AButton>
  <AButton medium>
    Button
  </AButton>
</>
  `}
/>

#### With Icons

To render a button with an icon, render `<AIcon />` to either the left or right of your text, and be sure to pass the direction in which the icon sits relative to the button text to `<AIcon />`

<Playground
  code={`<div className='d-flex align-center'>
  <AButton className="mr-2">
    <AIcon left>pencil-simple</AIcon>
    Edit
  </AButton>
  <AButton secondary className="mr-2">
    Save
    <AIcon right>save</AIcon>
  </AButton>
</div>`}
/>

#### Only With An Icon

To render a button with just an icon, pass the `icon` prop to the button to indicate that there is no text to represent the button content, and then render `<AIcon />` as a direct child.

<Playground
  code={`<>
  {["primary", "secondary", "tertiary", "tertiaryAlt"].map((variant) => {
    const variantProp = {
      [variant]: true,
    };
    return (
      <AButton icon {...variantProp} className="mr-2" key={variant}>
        <AIcon>sync</AIcon>
      </AButton>
    );
  })}
</>
`}
/>

#### Buttons as Links

If the `href` property is used, the component will render as an HTML anchor.

<Playground
  code={`<>
  <AButton href="https://www.cisco.com" className="mr-2">
    Launch Cisco.com
  </AButton>
  <AButton href="https://www.oldschool.runescape.com" secondary>
    Launch Nostalgia
  </AButton>
</>
`}
/>

## Component Props

## Router Integrations

Set the `component` property to your router's link type, for example:

```jsx
import {Link} from "next/link";

<AButton component={Link} href="/">
  Home
</AButton>;
```
