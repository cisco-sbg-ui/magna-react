# Modals

## Install

Integrate with your build to [auto-import](/#integrating), or add an import in your component:

```
import {AKeyValue, AKeyValueTable} from "@cisco-sbg-ui/magna-react";
```

## Usage (Table)

Emphasis: keys

Best to use for a long list or when vertical space is a concern

<Playground
  code={`
  <AKeyValueTable>
    <AKeyValue label="Data key">Data value</AKeyValue>
    <AKeyValue
      label="Data key 2"

    <AKeyValue label="Data key 3">Data value 3</AKeyValue>
