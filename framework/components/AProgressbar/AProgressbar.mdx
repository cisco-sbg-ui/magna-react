## Usage

### Import

<ComponentImport components="AProgressbar" />

### Usage

<Playground
  code={`() => {
  const [percentage, setPercentage] = useState(0);
  const [intervalObject, setIntervalObject] = useState();
  return (
    <div style={{width: 200}}>
      <h3>File1.txt</h3>
      <AProgressbar striped displayText percentage={percentage} />
      <h3>File2.txt</h3>
      <AProgressbar striped displayText percentage={percentage - 20} />
      <AButton
        onClick={() => {
          clearInterval(intervalObject);
          setPercentage(0);
          setIntervalObject(
            setInterval(
              () => setPercentage((percentage) => percentage + 20),
              1000
            )
          );
        }}>
        Upload
      </AButton>
    </div>
  );
}
`}
/>

## Accessibility Notes

By default, `AProgressbar` components are assigned the [WAI-ARIA](https://www.w3.org/WAI/standards-guidelines/aria/) role of [progressbar](https://www.w3.org/TR/wai-aria/#progressbar).

The `AProgressbar` component uses the [aria-valuemin](https://www.w3.org/TR/wai-aria/#aria-valuemin), [aria-valuemax](https://www.w3.org/TR/wai-aria/#aria-valuemax), and [aria-valuenow](https://www.w3.org/TR/wai-aria/#aria-valuenow) attributes to communicate the progressbar's progression.
