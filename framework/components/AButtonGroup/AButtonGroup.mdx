## Usage

### Import

<ComponentImport components="AButtonGroup" />

### Usage

<Playground
  withTwoPanes
  code={`function Example() {
  const [selectedValues, setSelectedValues] = useState(["One"]);
  return (
    <AButtonGroup
      selectedValues={selectedValues}
      onChange={(_, selected) => setSelectedValues(selected)}
    >
      <AButton value="One">One</AButton>
      <AButton value="Two">Two</AButton>
      <AButton value="Three">Three</AButton>
    </AButtonGroup>
  );
}`}
/>

#### Sizes

To set sizing for a button group, size each button individually like you would for `AButton`.

<Playground
  code={`function Example() {
  const [selectedValues, setSelectedValues] = useState(["pie"]);
  return (
    <>
      <AButtonGroup
        selectedValues={selectedValues}
        onChange={(_, selected) => setSelectedValues(selected)}
      >
        <AButton small value="One">One</AButton>
        <AButton small value="Two">Two</AButton>
        <AButton small value="Three">Three</AButton>
      </AButtonGroup>
      <ADivider />
      <AButtonGroup
        selectedValues={selectedValues}
        onChange={(_, selected) => setSelectedValues(selected)}
      >
        <AButton medium value="One">One</AButton>
        <AButton medium value="Two">Two</AButton>
        <AButton medium value="Three">Three</AButton>
      </AButtonGroup>
    </>
  );
}
`}
/>
