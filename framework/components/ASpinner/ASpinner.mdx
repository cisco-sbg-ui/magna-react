# Spinner

## Install

Integrate with your build to [auto-import](/#integrating), or add an import in your component:

```
import { ASpinner } from "@cisco-sbg-ui/magna-react";
```

## Basic usage

<Playground
  code={`
<div>
  <div>simple spinner without label</div>
  <ASpinner/>
  <ADivider/>
  <div>spinner with label</div>
  <ASpinner>spinner label</ASpinner>
</div>
`}
/>
