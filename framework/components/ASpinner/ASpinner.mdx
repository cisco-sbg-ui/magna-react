# Spinner

## Install

Integrate with your build to [auto-import](/#integrating), or add an import in your component:

```
import { ASpinner } from "@cisco-sbg-ui/magna-react";
```

## Usage
Spinners provide feedback to users while a process, action or content is being loaded, and are used when the specific duration or progress cannot be estimated or determined.
### When to use Spinner
 - When the load time of a process is more than 300 ms.
 - For user-initiated processes such as saving or submitting a form, applying filters, or asynchronous updates on a part of a page (section or inline content).
 - When the duration or completion status of the process cannot be estimated (indeterminate).
### When not to use Spinner
 - When load time is 300 ms or shorter, just show the data once it is loaded.
 - For initial page load or full page refreshes, use [Skeleton](/components/skeleton).
 - When the duration or progress can be estimated, use a [Progress Bar](/components/progressbar).

## Spinner label
Consider adding a label to communicate useful information about the process being performed.
 - Use words or phrases that concisely describe the process.
 - Avoid using “Loading” as it does not provide useful information about the process.

`}
/>
