# Drawers

## Install

Integrate with your build to [auto-import](/#integrating), or add an import in your component:

```
import {ADrawer} from "@cisco-sbg-ui/magna-react";
```

By default, `ADrawer` renders over the entire page as a Modal. In this sense, the document `body` is acting as the drawer's container.

You can change this behavior to have `ADrawer` render inside your own specified container.
