# Lists

## Install

Integrate with your build to [auto-import](/#integrating), or add an import in your component:

```
import {
  AList,
  AListItem,
  AListItemAction
  AListItemAvatar,
  AListItemContent,
  AListItemDivider,
  AListItemGroup,
  AListItemSubtitle,
  AListItemTitle,
} from "@cisco-sbg-ui/magna-react";
```

## Router Integrations

On `AListItem`, set the `component` property to your router's link type, for example:

```
// With react-router:
<AListItem component={RouterLink} to="/">List Item 1</AListItem>
```

## Accessibility Notes

The `AListItem` `click` event handler is triggered both by the `click` event and by the `keyDown` event for the `Enter` key.
