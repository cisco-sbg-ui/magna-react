# Skeleton

## Install

Integrate with your build to [auto-import](/#integrating), or add an import in your component:

```
import {ASkeleton, ASkeletonHeader, ASkeletonText, ASkeletonBlock} from "@cisco-sbg-ui/magna-react";
```
