## Accessibility Notes

`ASlider` follows the [WAI-ARIA](https://www.w3.org/WAI/standards-guidelines/aria/) recommendations
for [slider](https://www.w3.org/TR/wai-aria/#slider), with appropriate `role`, `aria-valuemin`,
`aria-valuemax`, `aria-valuenow`, `aria-readonly` attributes on the slider thumbs.

Each slider thumb on `ASlider` has the following keyboard event handlers:

- arrow-right, arrow-up: increases slider value one step.
- arrow-left, arrow-down: decreases slider value one step.
- page-up: increase slider 10 steps.
- page-down: decrease slider 10 steps.
- home: decrease slider to minimum.
- end: increase slider to maximum.
