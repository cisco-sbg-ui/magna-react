# Checkboxes

## Install

Integrate with your build to [auto-import](/#integrating), or add an import in your component:

```
import {ACheckbox} from "@cisco-sbg-ui/magna-react";
```

## No label wrapper

For accessibility, pass the label as `children`. When `withLabel` is set to false, the checkbox will _not_ be wrapped in the `<label`> component, but it will be set as `aria-labeledby`.

<Playground
  code={`() => {
  const [checked, setChecked] = useState(false);

return (

<>
  <div style={{minHeight: "3rem"}}>
    <label htmlFor="noLabel">Label for checkbox</label>
    <ACheckbox
      className="mr-2"
      withLabel={false}
      id="noLabel"
      checked={checked}
      onClick={(e) => {
        setChecked(!checked);
      }}
    />
  </div>
</>
); } `} />

## Accessibility Notes

By default, `ACheckbox` components are assigned the [WAI-ARIA](https://www.w3.org/WAI/standards-guidelines/aria/) role of [checkbox](https://www.w3.org/TR/wai-aria/#checkbox).

The `ACheckbox` component uses the [aria-checked](https://www.w3.org/TR/wai-aria/#aria-checked) attribute to express the input checked state.
The `ACheckbox` component uses the [aria-disabled](https://www.w3.org/TR/wai-aria/#aria-disabled) attribute to express the input disabled state.
