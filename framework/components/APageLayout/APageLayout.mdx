## Usage

### Import

<ComponentImport components="APageContainer, APageHeader, APageTitle, APageDescription, APageLabel,  APageTitleContent, APageTitleContentLeft, APageTitleContentRight, APageTitleTexttitle: AAccordion" />

### Usage

#### Terminology

- Top line - breadcrumbs and actions that can impact the entire page
- Page title - may contain tags, filters, loading status, and page-level actions like “Add XXX”
- Page description - Key top-level information to highlight
- Page label like MAC details - can be placed next to primary tabs
