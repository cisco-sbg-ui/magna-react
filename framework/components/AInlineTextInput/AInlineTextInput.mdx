### Import

<ComponentImport components="AInlineTextInput" />

### Usage

`AInlineTextInput` enters edit mode on click, and pressing the <kbd>Escape</kbd> key escapes edit mode and resets the input back to what was last enetered.

<Playground
  withTwoPanes
  code={`function Example() {
  const [value, setValue] = useState(\n\t\t"Click to edit"\n\t);
  return (
    <div>
      <AInlineTextInput
        value={value}
        onChange={(newValue) => {
          setValue(newValue);
        }}
      />
      <br />
      <i
        className='mds-neutral--neutral-10--text'
      >
        Actual Value: {value}
      </i>
    </div>
  );
}
`}
/>

#### Raw Input Configuration

If needed, you can pass an object to the `inputComponentProps` prop to send props down directly to the `<ATextInput />` component.

Below, this is demonstrated by setting a `prependIcon` key for the `<ATextInput />`.

<Playground
  code={`() => {
  const [value, setValue] = useState("Click to edit");
  return (
    <div>
      <AInlineTextInput
        required
        value={value}
        onChange={(newValue) => {
          setValue(newValue);
        }}
        inputComponentProps={{
          prependIcon: "tag"
        }}
      />
      <br />
      <i className='mds-neutral--neutral-10--text'>
        Actual Value: {value}
      </i>
    </div>
);
}
`}
/>

## Accessibility Notes

Interactable prepended and appended icons are tab-focusable with a visual overlay, as well as assigned the [WAI-ARIA](https://www.w3.org/WAI/standards-guidelines/aria/) role of [button](https://www.w3.org/TR/wai-aria/#button).
