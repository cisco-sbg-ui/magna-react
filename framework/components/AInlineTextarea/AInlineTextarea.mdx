## Usage

### Import

<ComponentImport components="AInlineTextarea" />

### Usage

`AInlineTextarea` enters edit mode on click, and pressing the <kbd>Escape</kbd> key escapes edit mode and resets the input back to what was last enetered.

<Playground
  withTwoPanes
  code={`function Example() {
  const [value, setValue] = useState(\n\t\t"Click to edit"\n\t);
  return (
    <div>
      <AInlineTextarea
        value={value}
        onChange={(newValue) => {
          setValue(newValue);
        }}
      />
      <br />
      <i
        className='mds-neutral--neutral-10--text'
      >
        Actual Value: {value}
      </i>
    </div>
  );
}
`}
/>

#### Raw Input Configuration

If needed, you can pass an object to the `inputComponentProps` prop to send props down directly to the `<ATextarea />` component.

Below, this is demonstrated by setting a `hint` key for the `<ATextarea />`.

<Playground
  code={`() => {
  const [value, setValue] = useState("Click to edit");
  return (
    <div>
      <AInlineTextarea
        required
        value={value}
        onChange={(newValue) => {
          setValue(newValue);
        }}
        inputComponentProps={{
          hint: "Clicking your escape key will cancel the edits you make."
        }}
      />
      <br />
      <i className='mds-neutral--neutral-10--text'>
        Actual Value: {value}
      </i>
    </div>
);
}
`}
/>
