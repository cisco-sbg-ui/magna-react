# Text Inputs

## Install

Integrate with your build to [auto-import](/#integrating), or add an import in your component:

```
import {AInlineTextarea} from "@cisco-sbg-ui/magna-react";
```

## Usage

`AInlineTextarea` enters edit mode on click.

Escape key when editing resets value to last entered.

<Playground
  code={`() => {
  const [value, setValue] = useState("inline edit value");
  return (
    <div>
      <AInlineTextarea value={value} onChange={(newValue) => {
        setValue(newValue);
      }} />
      <br />
      Actual Value: {value}
    </div>

);
}
`}
/>
